1 前言

众所周知，墙是越砌越高封锁连连，实在招架不住频繁更换服务器，无奈之下只好采用最新的技术来实现科学上网。由于是新鲜事物，并不像之前 SSR 有那么丰富的图形界面可供操作，还请诸位看官多多包涵。

2 PC端下载以及配置

Windows端 点这里 -> trojan-1.15.1-win.zip

Mac端 点这里 -> trojan-1.15.1-macos.zip

需注意在Windows运行该程序需要VC环境，如果打开报错请运行压缩包所包含的VC_redist64

如下是配置文件，两端都通用且压缩包已包含该文件config.json，请用文本编辑器打开并按照下面所标识的#位置处修改参数，要改的只有两处其他保持原封不动。

{
    "run_type": "client",
    "local_addr": "127.0.0.1",
    "local_port": 1080,
    "remote_addr": "example.com", # 此处更改为服务器域名
    "remote_port": 443,
    "password": [
        "******" # 此处更改为账号密码
    ],
    "log_level": 1,
    "ssl": {
        "verify": true,
        "verify_hostname": true,
        "cert": "",
        "cipher": "ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256:ECDHE-ECDSA-CHACHA20-POLY1305:ECDHE-RSA-CHACHA20-POLY1305:ECDHE-ECDSA-AES256-GCM-SHA384:ECDHE-RSA-AES256-GCM-SHA384:ECDHE-ECDSA-AES256-SHA:ECDHE-ECDSA-AES128-SHA:ECDHE-RSA-AES128-SHA:ECDHE-RSA-AES256-SHA:DHE-RSA-AES128-SHA:DHE-RSA-AES256-SHA:AES128-SHA:AES256-SHA:DES-CBC3-SHA",
        "cipher_tls13": "TLS_AES_128_GCM_SHA256:TLS_CHACHA20_POLY1305_SHA256:TLS_AES_256_GCM_SHA384",
        "sni": "",
        "alpn": [
            "h2",
            "http/1.1"
        ],
        "reuse_session": true,
        "session_ticket": false,
        "curves": ""
    },
    "tcp": {
        "no_delay": true,
        "keep_alive": true,
        "reuse_port": false,
        "fast_open": false,
        "fast_open_qlen": 20
    }
}

修改好就直接运行程序，Windows打开trojan.exe，Mac打开start.command

3 Chrome 搭配 SwitchyOmega 科学上网

由于没有像 SSR 有 Pac 代理或者全局代理，所以必须得用 Chrome + SwitchyOmega 才能正常走代理访问网站。

3.1 下载安装

Chrome 下载点这里 -> Chrome

SwitchyOmega 下载点这里 -> SwitchyOmega

运行 Chrome 找到扩展程序的选项，Windows在[更多工具]->[扩展程序]，Mac在[窗口]->[扩展程序]

把下载好的SwitchyOmega_Chromium.crx重命名SwitchyOmega_Chromium.zip，然后拖拽到扩展程序的这个页面，个别时候会收到浏览器安全提醒，忽略即可
